Structural Poverty and Labor Markets in Brazil

By Gabriel Costa | LSE Master of Public Policy

Aims

This project investigates the effectiveness of employability as a driver for poverty reduction. By analyzing Maranhão and Piauí, I test if labor market shocks can overcome deep structural deficits.

Data

Primary data is sourced from the IBGE SIDRA API. I utilize GeoJSON for spatial mapping and JSON time-series for policy evaluation.

Chart 4: Advanced Interactive Exploration

View Data

Interactivity: This map provides structural context for the analysis by showing poverty and sanitation conditions across Brazilian states in 2022. In addition to absolute levels, the visualization allows values to be expressed relative to the national average, highlighting regional disparities. This perspective helps explain why income transfer programs such as Bolsa Família remain particularly important in historically disadvantaged regions, especially in the Northeast.

Chart 6: Structural deviations from the national average

View Data (JSON)

This chart standardizes each deprivation indicator into a z-score relative to the national mean. Values to the left of zero indicate worse structural conditions than the Brazilian average; values to the right indicate better conditions. To ensure consistent interpretation across indicators, the unemployment z-score is sign-inverted so that higher values always correspond to better outcomes. Maranhão is highlighted.

Tools

Data processing performed in Python (Pandas). Interactive visualizations built using Vega-Lite and hosted via GitHub Pages.

Conclusions

Evidence suggests that isolated employability shocks are insufficient without aggressive investments in basic infrastructure and human capital.
